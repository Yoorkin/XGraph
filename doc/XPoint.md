# XPoint

坐标集合

## 成员
| 成员 | 返回类型 | 描述 |
| :---: | :---: | :-:|
| x | Long | x坐标值|
| y | Long | y坐标值|
| Clone | XPoint | 克隆信息相同的的副本 |

## 创建方法

```
Dim P as XPoint
Set P = XGraph.Pos(100,200)
```

***tip:所有的创建方法均位于`XGraph`全局空间中,使用创建方法时可不加`XGraph.`***
