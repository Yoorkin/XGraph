# FontStyle
```
Public Enum FontStyle
    FontStyleRegular = 0    '默认
    FontStyleBold = 1       '加粗
    FontStyleItalic = 2     '斜体
    FontStyleBoldItalic = 3 '加粗和斜体
    FontStyleUnderline = 4  '下划线
    FontStyleStrikeout = 8  '删除线
End Enum
```

***tip:使用如`FontStyleBold + FontStyleUnderline`的方式可将多个样式组合在一起***
