# XBounds
宽高和xy信息的集合
## 成员

| 成员 | 返回类型 | 描述 |
| :---: | :---: | :-:|
| x | Long | x坐标值|
| y | Long | y坐标值|
| Width | Long | 宽度|
| Heght | Long | 高度|
| Clone | XBounds | 克隆信息相同的的副本 |

## 创建方法

```
Dim b as XBounds
Set b = XGraph.Bounds(10,10,100,200)
```
