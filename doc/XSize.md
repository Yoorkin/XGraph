# XSize

宽高信息的集合
## 成员

| 成员 | 返回类型 | 描述 |
| :---: | :---: | :-:|
| Width | Long | 宽度|
| Heght | Long | 高度|
| Clone | XSize | 克隆信息相同的的副本 |

## 创建方法

```
Dim s as XSize
Set s = XGraph.Size(100,200)
```

***tip:所有的创建方法均位于`XGraph`全局空间中,使用创建方法时可不加`XGraph.`***
